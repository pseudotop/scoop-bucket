# Scoop Bucket for ONESHIM

Official [Scoop](https://scoop.sh) bucket for ONESHIM tools.

## Installation

```powershell
scoop bucket add oneshim https://github.com/pseudotop/scoop-bucket
scoop install oneshim
```

## Update

```powershell
scoop update oneshim
```

## Available Apps

| App | Description |
|-----|-------------|
| `oneshim` | AI-assisted office productivity desktop client |
